F.L.Y Entertainment — Facebook Ads Practice Campaign
Project Overview

F.L.Y Entertainment is a digital marketing and entertainment branding project developed as part of a Digital Marketing Internship program. The project focuses on creating and managing a professional Facebook and Instagram advertising campaign using Meta Ads Manager while building a modern entertainment brand inspired by global K-pop culture and performance-based entertainment industries.

The main objective of this project was to gain practical experience in social media marketing, audience targeting, campaign optimization, advertisement strategy, and digital brand promotion through real-world Meta advertising workflows.

Project Objectives
Create a professional entertainment brand identity
Develop and manage Facebook & Instagram ad campaigns
Improve brand awareness and audience engagement
Analyze advertisement performance using Meta Ads Manager
Understand campaign optimization and audience targeting
Build a responsive entertainment website for brand promotion
